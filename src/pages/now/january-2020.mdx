# Now

I try to keep what I'm working on and reading to a minimum in order
to make sure I don't get caught multi-tasking.

If you're interested in what I'm currently up to in January 2020,
here's a brief summary.

## Working

- [Blocks](https://blocks-ui.com): A JSX-based page builder for creating beautiful websites without writing code.
- [MDX](https://mdxjs.com): An [authorable format](/authorable-format) that lets you seamlessly write JSX in your Markdown documents.
- [Components AI](https://components.ai): Experimental computational design platform for exploring generative space.
- [Sector](https://sector.tools): An MDX-based writing platform

## Writing/teaching

- [Compilers for Humans](https://compilersforhumans.com): A course that seeks to show folks that compilers aren't magic.
- [egghead lessons](https://egghead.io/instructors/john-otander): Recording screencasts around ASTs, MDX, and Gatsby.
