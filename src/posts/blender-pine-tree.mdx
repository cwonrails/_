The first thing I did was select the "general" document
and I was greeted with a cube on the canvas and a vast
array of tools and tabs.

![image](https://user-images.githubusercontent.com/1424573/99978457-35a5c280-2d63-11eb-915a-5eb4e22f7d2e.png)

At this point I don't even know where to start. Time
to google. But, first I need to figure out what I want
to make. My head goes to a tree.

### Making a tree

I found a [tutorial](https://blender.stackexchange.com/questions/8688/is-there-an-easy-way-to-make-a-tree-in-blender)
that uses the [Sapling Tree Gen addon](https://docs.blender.org/manual/en/latest/addons/add_curve/sapling.html).
I'm not always super into going straight into addons
when learning new software, but at the same time getting
something on the canvas quickly seemed worthwhile.

#### Enable the Sapling addon

Edit > Preferences > Addons > Filter for "Sapling" > Check the box

![image](https://user-images.githubusercontent.com/1424573/99979409-4c98e480-2d64-11eb-89e9-d6ebf568a937.png)

3D Viewport > Add > Curve > Sapling Tree Gen

![image](https://user-images.githubusercontent.com/1424573/99979804-bfa25b00-2d64-11eb-8161-fbe3b3491e19.png)

And now we have a tree inside the cube.

![image](https://user-images.githubusercontent.com/1424573/99979968-eeb8cc80-2d64-11eb-8261-1650d2942426.png)

I then hid the cube but had no idea where to go from there.
[Time for some YouTube](https://www.youtube.com/watch?v=Ojg3dV1cJko)

From the video I learned I could use `x` to delete cube.
Then I used the pine tree preset.
