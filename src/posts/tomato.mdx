**AAA** score and the contrast ratio is **7.1**.

</Flex>

<Flex>

<div style={{ height: 200, width: 200, minWidth: 200, flexGrow: 1, color: 'tomato', border: '2px solid tomato', fontWeight: 'bold', display: 'flex', alignItems: 'center', justifyContent: 'space-around', lineHeight: 1, fontSize: 80, marginRight: 20 }}>F</div>

When combined with `white`, `tomato` isn't accessible. It results in an
**F** score and the contrast ratio is **2.9**. At first this might seem
like a bummer, but we can still use it for accents like borders.

</Flex>

<Flex>

<div style={{ height: 200, width: 200, minWidth: 200, flexGrow: 1, border: '16px solid tomato', backgroundColor: 'black', color: 'tomato', fontWeight: 'bold', display: 'flex', alignItems: 'center', justifyContent: 'space-around', lineHeight: 1, fontSize: 80, marginRight: 20 }}><code>#</code></div>
